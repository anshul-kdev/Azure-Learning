Self-Service Password Reset (SSPR)

Evaluate and implement SSPR to allow users in your organization to reset their passwords or unlock their accounts without help desk intervention.

Prerequisite Warning: Enabling SSPR for end-users requires at least a Microsoft Entra ID P1 or P2 license assigned to the tenant. Free tier users can only test this as a global administrator.

Exercise 1: Enable Self-Service Password Reset

The first step is to activate the feature in your directory. You can choose to roll this out to a specific test group first, or enable it for all users.

  1. Sign in to the Azure portal as a Global Administrator.
  2. Search for and select Microsoft Entra ID.
  3. In the left-hand navigation menu, scroll down to the Manage section and select Password reset.
  4. On the Properties page, under Self service password reset enabled, choose either Selected (and pick a specific security group) or All.
  5. Click Save at the top of the page.

Exercise 2: Configure Authentication Methods

Users need a way to prove who they are before resetting their password. In this step, we configure which methods are acceptable for verification.

  1. Still inside the Password reset menu, select Authentication methods from the left side.
  2. Set the Number of methods required to reset to 1 (or 2 for higher security).
  3. Under Methods available to users, check the boxes for:
    • Mobile app notification
    • Mobile app code
    • Email
    • Mobile phone
  4. Click Save to apply these authentication methods.
  5. (Optional) Go to the Registration tab and ensure "Require users to register when signing in" is set to Yes. This forces users to set up their phone/email the next time they log in.

Exercise 3: Customize Directory Branding

To ensure users trust the password reset portal, you should customize the sign-in page to display your organization's logo and helpdesk information.

  1. Navigate back to the main Microsoft Entra ID overview page.
  2. Under the Manage section, select Company branding.
  3. Click Edit on the default branding or click + Customize to create a new one.
  4. Upload a Sign-in page background image (1920x1080px) and a Banner logo (280x60px).
  5. Click the Review + save button, then Save.
Verification: Open a new private/incognito window and navigate to https://aka.ms/ssprsetup to see the registration portal, or go to https://passwordreset.microsoftonline.com/ and verify your new custom branding appears!
