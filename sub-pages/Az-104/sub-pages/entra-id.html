MS Learn Module 2 Practical

Create, Configure, and Manage Identities

Hands-on exercises for managing users, groups, and licenses in Microsoft Entra ID.

Note on Curriculum: MS Learn Module 1 ("Understand Microsoft Entra ID") focuses entirely on theoretical concepts (comparing AD DS, P1/P2 pricing). We have skipped directly to Module 2 to begin the practical, hands-on lab exercises.

Exercise 1: Create Users and Assign Licenses

Organizations need a definitive identity for each user to control access to cloud workloads. First, we will create a user and assign them a license to use Azure services.

  1. Sign in to the Azure portal using your administrator account.
  2. In the global search bar, search for and select Microsoft Entra ID.
  3. In the left menu, under Manage, select Users, and then select All users.
  4. At the top of the page, select + New user > Create new user.
  5. Under the Basics tab, enter a User principal name (e.g., chris@yourtenant.onmicrosoft.com) and Display name (e.g., Chris Green).
  6. Check the Auto-generate password box and copy the password to your clipboard. You will need this to test the login later.
  7. Click Review + create, and then Create.
  8. Assigning the License: Go back to All users, click on Chris Green. On the left menu under Manage, click Licenses. Click + Assignments, select an available license (like Microsoft Entra ID P2 or Microsoft 365), and click Save.
Verification: Open a private/incognito browser window. Go to portal.azure.com and log in with Chris's new UPN and auto-generated password. You will be prompted to change the password on the first login!

Exercise 2: Add Groups and Group Licenses

Assigning licenses to individual users is tedious. Best practice dictates that we assign licenses to a Group, and simply add users to that group.

  1. Return to the Microsoft Entra ID overview page.
  2. Under Manage, select Groups.
  3. Click + New group at the top.
  4. Set the Group type to Security.
  5. Enter a Group name: Cloud Administrators and a description.
  6. Set Membership type to Assigned.
  7. Click the No members selected link. Search for Chris Green, click his name to add him to the "Selected" section, and click Select.
  8. Click Create.
  9. Group Licensing: Open the newly created Cloud Administrators group. Click Licenses > + Assignments, select a license, and save. Chris Green will now automatically inherit this license via group membership.

Exercise 3: Restore Deleted Users

When a user is deleted in Entra ID, they are placed in a "Deleted users" container for 30 days. During this window, you can restore their account and all their access.

  1. Go to Microsoft Entra ID > Users > All users.
  2. Search for Chris Green. Check the box next to his name, click the Delete icon at the top menu, and confirm.
  3. On the left-hand menu, under Manage, click Deleted users.
  4. You will see Chris Green listed here. Check the box next to his name.
  5. At the top of the page, click Restore user and confirm.
Verification: Go back to All users. Chris Green will reappear with all his previous group memberships and licenses fully intact.
