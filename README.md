# Azure Learning ☁️

Welcome to the **Azure-Learning** repository! This project serves as a centralized hub for documenting my technical journey, notes, and practical implementations as I explore the Microsoft Azure cloud ecosystem.

## 📌 Overview

This repository is the codebase for various hands-on labs and configurations. For a more structured, readable, and detailed breakdown of the concepts, please visit the companion documentation site:

👉 **[WEB Preview](https://anshul-kdev.github.io/Azure-Learning/)**

The website provides a step-by-step walkthrough of the learning path, featuring conceptual deep-dives and detailed explanations of the services implemented here.

## 🚀 Practical Implementations

This repository focuses on the practical application of Azure services. The content covers various domains, including:

* **Compute:** Deploying and managing Virtual Machines, App Services, and Serverless Functions.
* **Networking:** Implementing Virtual Networks (VNet), Subnets, and Network Security Groups (NSG).
* **Storage & Databases:** Hands-on work with Blob Storage, Azure SQL, and Cosmos DB.
* **Identity & Security:** Managing access via Azure Active Directory (Microsoft Entra ID) and securing secrets in Key Vault.
* **Infrastructure as Code (IaC):** Automating deployments using ARM templates, Bicep, or Terraform.

## 🛠️ Getting Started

To follow along with the practicals:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/anshul-kdev/Azure-Learning.git](https://github.com/anshul-kdev/Azure-Learning.git)
    ```
2.  **Practice the content:** Visit the [GitHub Pages site](https://anshul-kdev.github.io/Azure-Learning/) to find the context and instructions.
3.  **Prerequisites:** You will generally need an active Azure subscription.
